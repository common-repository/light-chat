== Description ==

A very light and simple chat plugin.

Features:
Only sends ajax request when there are new messages,
Takes up the CSS of your wordpress theme


== Installation ==
1. Upload `light-chat.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You should be able see and use the "Light Chat" widget.
4. For comments and suggestions visit: http://light-chat.cdobiz.com/
